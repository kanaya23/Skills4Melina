## File Send Rules

- Any file meant to be uploaded back to Discord must be saved under `workspace/Melina/out/`.
- Never send files directly from `/tmp`.
- If a file is created in `/tmp`, copy it into `workspace/Melina/out/` before sending.
- Name output files clearly and predictably.
- Prefer preserving the original extension unless conversion is explicitly useful.
- Before sending a file, verify that it exists and is readable.
